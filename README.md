# Lab 5: A Basic Math Calculator

This Java program provides a command-line calculator with options for basic arithmetic operations, modulus, and factorial calculation. 

## Highlights
- Performs addition, subtraction, multiplication, division, modulus.
- Includes factorial computation.
- User-friendly command-line interface.
- Error handling for invalid input.

## Overview

Lab5 is a simple console application designed to perform common mathematical calculations. Users can choose from various operations: addition, subtraction, multiplication, division, modulus, or factorial. The program handles both integer and factorial calculations.  It employs basic input validation to ensure only integers are used in calculations and provides user-friendly messages for incorrect input.

## Why This Project
While there are many advanced calculators available, Lab5 aims to provide a minimalistic tool for learning basic programming concepts and practicing input/output operations, conditional statements, and loops. It serves as a foundation for exploring more complex mathematical applications in Java. 

## Usage
Here's how to use Lab5:
```
1. Compile the Lab5.java code using a Java compiler (e.g., 'javac Lab5.java' ).
2. Run the compiled program (e.g., 'java Lab5').
3.  The program will display a menu of options. Choose an option by entering the corresponding number.
4. Follow the prompts to enter the required numbers for your calculation.

```


## Installation
- You need Java Development Kit (JDK) installed on your system. 
- Compile 'Lab5.java' using a command line: `javac Lab5.java`
- Run the compiled program: `java Lab5`

## Architecture
The project consists of a single Java class, "Lab5," containing the following parts:

*  `main()` method: The entry point of the program, handling user input, calculating results, and managing the loop for interaction.

* Helper Methods (`addition`, `subtraction`, `multiplication`, `division`, `modulus`, `factorial`): These methods perform individual arithmetic operations.  

**File Structure:**
- `Lab5.java`: Contains all the code for the calculator.


## Security Notes 
This project does not involve sensitive data or security considerations. User input is immediately used in calculations and not stored persistently.
